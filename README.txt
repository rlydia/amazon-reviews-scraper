用到Selenium自动化


.py文件中与三处修改的地方，已在文件中标注（皆位于 初始化 中）：

1）必须修改替换：
self.browser.get(" ")中替换为所需爬取商品的评论页面链接

2）必须修改替换
self.ASIN = ''替换为所爬取商品的ASIN
self.CsvFileName = 'test.csv' 替换为自定义文件名称

3）可修改替换
self.CsvN = 用于计数--当前爬取页数

注意：国内网很容易中断，用VPN有时也会不稳定；若中断后，将self.browser.get(" ")中的链接替换为当前断开的页数链接。之后继续爬取，内容会接着在csv文件中继续写入，但是 设置csv文件首行的代码 会再次执行一次，可以把这段代码注释掉，或者不用管它，之后会再去重中去掉。
